üì§ Blog Deployment Overview

A quick recap of how Hexo + GitHub Pages works:

  1. Write Markdown posts

  2. Hexo generates static HTML files

  3. hexo server previews the site locally

  4. hexo deploy pushes it to GitHub

  5. GitHub Pages hosts your blog at https://yourusername.github.io

Error Solution
hexo: command not found Install Hexo globally: npm install -g hexo-cli
Permission denied (publickey) Check SSH key, ensure it‚Äôs added to GitHub
Cannot find module Run npm install to install dependencies
Blog doesn‚Äôt update Run hexo clean before generating
GitHub Pages 404 Check repo name and branch, and wait a few minutes
